# Portfolio Hero (React + Vite)

A React + Vite single-page hero that mirrors the look and feel of aayushbharti.in: neon grid background, bold gradient typography, and CTA buttons.

## Getting Started

Prerequisites: Node.js 18+ and npm.

```bash
npm install
npm run dev
```

Visit the printed local URL to view the page.

## Production Build

```bash
npm run build
npm run preview
```

The build output lives in `dist/`.
