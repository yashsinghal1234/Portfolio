# Portfolio (React + Vite)

A React + Vite portfolio with a neon grid aesthetic, curated work section, interactive globe, and an about section with profile image and social links.

## Getting Started

Prerequisites: Node.js 18+ and npm.

```bash
npm install
npm run dev
```

Visit the printed local URL to view the page.

## Features

- Hero section with CTA buttons and highlighted resume link
- Skills grid with branded tech logos
- Curated work with scrolling cards and sticky details
- Interactive globe for time zone selection
- About section with profile image and social links

## Production Build

```bash
npm run build
npm run preview
```

The build output lives in `dist/`.
